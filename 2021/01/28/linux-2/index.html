文件管理

ls

列出指定目录下的所有文件，列出/目录下的文件

pwd

获取目前所在工作目录的绝对路径

cd

改变当前工作目录

date

显示或修改系统时间与日期

passwd

用于设置用户密码

su

改变用户身份（切换到超级用户）

clear

用于清除屏幕信息

man

显示指定命令的帮助信息

who

显示目前登录到系统的用户

free

显示系统内存状态（单位MB）

ps

显示系统进程运行动态

端口

查看所有应用占用端口情况

查找8080端口是否被占用

进程

查看所有进程

查看tomcat进程

（输出如下 *** 5144 。。。等等.Bootstrap start 说明tomcat已经正常启动， 5144 就为进程号 pid = 5144）

查看监听进程号

杀死进程

查看sshd进程的运行动态

top

查看即时活跃的进程，类似Windows的任务管理器

mkdir

创建目录

more

用于分页查看文件，例如每页10行查看boot.log文件

cat

用于查看文件，例如查看Linux启动日志文件文件，并标明行号

touch

用于创建文件，例如创建text.txt文件

rm

删除文件

强制删除某个目录及其子目录

cp

用于拷贝文件，例如将test1目录复制到test2目录

mv

用于移动或覆盖文件

压缩与解压

tar

将/etc文件夹中的文件归档到文件etc.tar（并不会进行压缩）

用gzip压缩文件夹/etc中的文件到文件etc.tar.gz

用bzip2压缩文件夹/etc到文件/etc.tar.bz2

分页查看压缩包中内容（gzip）

解压文件到当前目录（gzip）

解压文件到指定目录（gzip）

终端编辑器

vi/vim(需要安装)

打开/创建文件

编辑模式

按任意键开始编辑

正常模式

esc

保存、退出

shift + :

w–保存
q–退出
wq–保存并退出
q!–强制退出
set nu–显示行号
set nonu–隐藏行号

注意若非正常关闭vim，则会生成临时文件（隐藏的），需要删除

df

查看磁盘空间占用情

du

查看当前目录下的文件及文件夹所占大小

ifconfig

显示当前网络接口状态

netstat

查看当前路由信息

查看所有有效TCP连接

查看系统中启动的监听服务

查看处于连接状态的系统资源信息

wget

从网络上下载文件

文件上传下载

rpm

RPM是Red-Hat Package Manager的缩写，一种Linux下通用的软件包管理方式，可用于安装和管理.rpm结尾的软件包。

Yum是Yellow dog Updater, Modified的缩写，能够在线自动下载RPM包并安装，可以自动处理依赖性关系，并且一次安装所有依赖的软件包，非常方便！
